# Support

## Questions and bugs

- Use GitHub issues for reproducible bugs, documentation gaps, and feature
  requests.
- Include the command or workflow you ran, expected behavior, actual behavior,
  and relevant logs with secrets removed.

## Security issues

Do not open a public issue for security concerns. Follow `SECURITY.md` instead.

## Powerset-hosted services

Some workflows require a Powerset account or hosted API access. If a command
fails because you are not provisioned for a hosted integration, include the
error message in your issue without sharing tokens or private keys.
