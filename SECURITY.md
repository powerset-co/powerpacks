# Security Policy

## Reporting a vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Email security reports to security@powerset.co with:

- a description of the issue and affected files or commands;
- reproduction steps or proof-of-concept details, if available;
- the potential impact;
- any suggested remediation.

We will acknowledge reports within 3 business days and follow up with status or
next steps after triage.

## Supported versions

Security updates are handled on the `main` branch unless a release branch is
explicitly announced.
